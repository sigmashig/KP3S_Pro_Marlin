v.1.01 от 06.03.2023
Прошивка для 3D принтера KingRoon KP3S Pro от 04.03.2023 на базе Marlin (2.1.x-fixes).
Особенности прошивки:
 - Портретный режим работы экрана;
 - Поддержка русского языка;
 - Фикс бага SD карточки (отвал при попытке открыть файл);
 - Фикс бага паузы из gcode (M125), нельзя было продолжить печать после паузы;
 - Черный фон;
 - Включен Linear Advance;
 - Включен Input Shaping;
 - Включены автопрофили для ABS PLA PETG;
 - Поддержка смены филамента;
 - Калибровка экрана при первой загрузке;
 - Калибровка стола по 25 точкам;
 - Откалиброван шаг экструдера M92 c заводского значения 768 на значение 792 (+3мм); 
 - Возможность автокалибровки PID Hotend и стола из расширенного меню;
 - Из меню доступно редактирование всех основных параметров принтера.

Собрана прошивка: Igor Samatov (@stopservice).
